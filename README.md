
# Blog Website For Test

## Installation


```bash
  git clone https://github.com/ssahibsingh/blog-website.git
```

Go to the project directory

```bash
  cd blog-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node app.js
```
Change MongoDB Connection String

```bash
  Change Connection String in app.js
```
